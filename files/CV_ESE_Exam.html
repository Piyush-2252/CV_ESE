AI309E - ICV ESE EXAM

Blood Cell Classification

Basophil vs Erythoblast Classification using Computer Vision and Machine Learning

100 Total Marks

Overview

You are provided with a dataset containing images of two types of blood cells: Basophils and Erythoblasts. Your task is to build a complete machine learning pipeline for this binary classification.

The examination consists of Five parts:
  • Part A — Image Preprocessing
  • Part B — Feature Extraction
  • Part C — ML Pipeline Development
  • Part D — Comparitive Analysis
  • Part E — Quiz

Each task must be in a separate, clearly labelled code cell with concise comments. Display visualizations wherever applicable with proper labels and titles.

Important Warnings

  • Missing Steps: Skipping any Part completely will result in deduction of marks.
  • No Output Display: Failing to display outputs (images, plots, metrics) wherever necessary will result in deduction of marks .
  • Incomplete Pipeline: The ML pipeline must include all four models (Logistic Regression, SVM, KNN, Random Forest) — missing any model will result in proportional mark deduction.
  • Quiz Completion: The quiz is mandatory and must be attempted after completing the coding tasks.
